# moobook 시나리오 목록 (10개)

AI 기반 맞춤형 아동 동화책 서비스 **moobook**의 시나리오 데이터입니다.
실제 데이터 소스: `src/lib/scenarios/`

---

## 공통 사항

- 모든 시나리오는 **12페이지**로 구성
- `{childName}`은 실제 아이 이름으로 치환되며, `(이)는 / 이(가) / (아)` 등 조사 토글도 `lib/utils/korean-name.ts`의 `replaceChildName()`에서 받침 유무에 따라 자동 변환됨
- 일러스트 프롬프트에는 워터컬러 스타일 suffix가 자동 부여됨
- 카테고리(`ScenarioCategory`): `adventure | daily-life | emotion | celebration | science`

### 카테고리별 시나리오 분류

| 카테고리               | 시나리오                                                            |
| ---------------------- | ------------------------------------------------------------------- |
| 모험 (adventure)       | 숲속 대모험, 바다 친구들                                            |
| 일상생활 (daily-life)  | 양치 히어로, 목욕 대작전, 요리 마법사, 동물 학교                    |
| 감정/성장 (emotion)    | 첫 등원 날                                                          |
| 기념일 (celebration)   | 내 생일 대모험                                                      |
| 과학 (science)         | 우주 탐험대, 공룡 나라                                              |

### 시나리오 필드 구조

```ts
interface Scenario {
  id: ThemeId;
  title: string;
  description: string;
  category: ScenarioCategory;
  educationMessage: string;
  targetAge: string;
  pageCount: number;
  pages: ScenarioPage[];
}

interface ScenarioPage {
  pageNumber: number;
  text: string;              // 동화 본문 (아이 이름 자리 {childName})
  sceneDescription: string;  // 한국어 장면 설명
  illustrationPrompt: string;// 영문 AI 프롬프트 (+ 공통 스타일 suffix)
  emotion: string;           // 감정 태그
}
```
